******************* Compilation ******************* 
---------------------------------------------------

"make" puis "make run" pour lancer le programme avec les parametres par défaut (4 ateliers, 2 produits)

******************* Lancement *********************
---------------------------------------------------
./kanban

Après execution du programme, une interface de dialogue vous proposera de configurer les différent paramètre possible.

5 pour valider les paramètre par défault

********************** FAQ ************************
---------------------------------------------------
Q: Pourquoi l'affichage est un peut anarchique?
A: Les différent ateliers (Thread) travaille en même temps, ils affiche donc un message quand il effectue une action.
	Et ce peut importe ce qu'avait afficher le thread précédent
